# Projeto Cantina Senac

## Como Executar

1. Backend (Java + Spring Boot)
- Requisitos: Java 11+, Maven instalado
- Comando:
    cd senac-cantina
    mvn spring-boot:run
- A API ficará disponível em http://localhost:8080/produtos

2. Frontend (HTML, CSS, JS)
- Navegue até a pasta 'senac-cantina-frontend'
- Abra o arquivo 'index.html' no navegador
- Ele consumirá os dados da API automaticamente

Obs: Para visualizar corretamente as imagens, mantenha a pasta 'img' no mesmo local que o index.html.
